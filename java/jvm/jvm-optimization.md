# jvm优化

## 调优参数 
JVM 主要接受两类标志：布尔标志和附带参数的标志    
布尔标志: 
   -XX:+FlagName 表示开启， -XX:-FlagName 表示关闭    
附带参数的标志:
   -XX:FlagName=something
